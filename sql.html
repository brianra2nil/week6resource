There is no functionality in this code, the purpose is so it can be copied and pasted into workbench.

schema.sql file (replace with the variables and name of your database)


DROP DATABASE IF EXISTS todo_db;

CREATE DATABASE todo_db;

USE todo_db;

CREATE TABLE items (
id INT AUTO_INCREMENT NOT NULL PRIMARY KEY,
text VARCHAR(50) NOT NULL,
isDone BOOL NOT NULL DEFAULT 0
);
